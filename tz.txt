Важно правильно понять задачу:

1. НЕ нужно копировать чужой сайт-партнёр.
2. НЕ нужно брать контент, тексты, товары и структуру с worldcashboxvl.ru.
3. С сайта worldcashboxvl.ru нужно взять ТОЛЬКО контактные данные компании.
4. Сам сайт должен быть посвящён продуктам и решениям Эвотор.
5. Товары, фото товаров, названия, актуальные цены и базовые характеристики нужно брать с официального сайта Эвотор.
6. Визуально сайт должен быть выполнен в стилистике бренда Эвотор: чисто, технологично, современно, с фирменной цветовой логикой, но без прямого копирования официального сайта.
7. Это должен быть лендинг партнёра Эвотора, который приводит заявки на покупку оборудования и подключение услуг.

4. ВАЖНО ПО ДАННЫМ ТОВАРОВ
========================================

Нужно предусмотреть один из двух вариантов реализации:

Вариант A — предпочтительный:
- написать отдельный модуль/скрипт импорта товаров с официального сайта Эвотор
- скрипт должен собирать:
  - название
  - цену
  - URL товара
  - URL изображения
  - краткое описание
  - основные характеристики
- после этого данные сохранять в локальный JSON/TS-файл или другой удобный источник данных проекта

Вариант B:
- если полноценный парсинг в рантайме делать нецелесообразно, тогда:
  - заложить data layer для товаров
  - реализовать import script / parser stub
  - наполнить стартовые данные по официальному сайту Эвотор
  - оставить понятную архитектуру для последующего обновления цен и карточек

Важно:
- не парсить товары в клиенте браузера
- не делать зависимость рендера сайта от live-парсинга внешнего сайта
- лучше сделать один раз импорт и использовать локальные данные
- цены и изображения должны быть легко обновляемыми

50. Загрузить сделку в Saby CRM по API
Запрос создает сделку на основании данных из внешней системы. В результате запроса сделки из внешней системы будут перенесены в Saby CRM.

При загрузке сделки вы можете:

проверить, какие темы отношений есть в Saby, и выбрать нужную для загрузки;
проверить, есть ли ваш клиент в Saby и, если нет, создать его.
Параметры запроса 
Метод запроса: CRMLead.insertRecord

Адрес запроса: https://online.sbis.ru/service/
